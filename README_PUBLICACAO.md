# Site — Lavagem da Esquina do Padre

Site institucional estático, responsivo e pronto para publicação.

## Arquivos principais

- `index.html`: página do site
- `assets/`: imagens, estilos e scripts
- `portfolio.pdf`: portfólio institucional para leitura/download

## Publicação no Cloudflare Pages

1. Crie um projeto em **Workers e Pages**.
2. Escolha envio direto de arquivos ou conexão com repositório.
3. Publique a pasta inteira `site_lavagem_esquina_do_padre`.
4. Não há comando de compilação; o diretório de saída é a própria raiz.
5. Configure `esquinadopadre.com.br` como domínio principal.
6. Configure `lavagemdaesquinadopadre.com.br` para redirecionar ao domínio principal.
